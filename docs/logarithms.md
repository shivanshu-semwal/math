# Logarithms

## Definition

- $\log_b c = a \implies b^a = c, b \not= 0$

## Properties

- $\log m^n = n \log m$
- $\log (mn) = \log m  + \log n$
- $\log \frac{m}{n} = \log m - \log n$
- $\log_a 1 = 0$
- $\log_a a = 1$
- $\log_{a^b} m = \frac{1}{b}\log_a m$
- $\log_{a^b} m^n = \frac{n}{b}\log_a m$
- $\log_a b = \frac{\log b}{\log a}$
- $\log_a b = \frac{1}{\log_b a}$
- $a^{\log_a x} = x$
